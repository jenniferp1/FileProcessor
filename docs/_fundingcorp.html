_fundingcorp module¶

FileProcessor Mixin Class¶

Support FileProcessor Class.

Module contains Funding Corp data file specific processors. User supplies processor method name and a DataFrame with the data for processing.

Classes:
fileprocr.FileProcessor.FileProcessor - Loads and processes sets of flat files (ETL)

choose_processor(df, proc_info)[source]¶

Select user supplied processor method.

Available processing methods for:

  1. Average Balances TB Q?.xls

  2. Balance Sheet TB Q?.xls

Parameters

  • df (pd.DataFrame) – DataFrame with data needing processing.

  • proc_info (dict) – Dictionary containing method name.

Returns

Processed DataFrame ready for upload to data warehouse.

Return type

pd.DataFrame

Examples

Sample dictionary format for proc_info (must include a method key as shown).

{"class":"_fundingcorp", "method":"avg_bal_tb"}

Notes

Users need to add new processors methods if additional Funding Corp data files are provided.
